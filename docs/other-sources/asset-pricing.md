# Asset Pricing

- <a href="https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html" target="_blank">Kenneth French Data Library</a> — Fama/French factor returns and research portfolios (3-factor, 5-factor, momentum, industry portfolios, and more). Downloadable as CSV. Updated monthly. Also available via WRDS with Python API access — see our [Fama-French WRDS guide](../wrds/databases/fama-french.md). Also accessible in Python/R via community packages.
- <a href="https://www.openassetpricing.com/data/" target="_blank">Open Source Asset Pricing</a> — Replicated portfolio returns and stock-level signals for 200+ cross-sectional predictors from the academic literature, by Chen & Zimmermann (2022, *Critical Finance Review*). Downloadable as CSV and accessible via Python package. (<a href="https://github.com/OpenSourceAP/CrossSection" target="_blank">GitHub</a>)
- <a href="https://openbondassetpricing.com/" target="_blank">Open Source Bond Asset Pricing</a> — Replication data and code for corporate bond factors, including daily TRACE bond panels, bond-Compustat/CRSP links, and 341 bond/stock predictors. By Dickerson, Mueller & Robotti (2023). (<a href="https://openbondassetpricing.com/code/" target="_blank">Code</a>)
